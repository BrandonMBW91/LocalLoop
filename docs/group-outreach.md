## Notes / playbook (learned Jul 9, 2026)
- Space posts out (a few per day, not all at once) so Facebook doesn't flag spam.
- No App Store links in posts/comments — use the "search Local Loop" text CTA.
  Exception: B2B groups get the plain-text localloop.io/advertise URL (conversion
  beats reach with buyers).
- Post archetypes: weekend/week roundup (default) · nostalgia angle (memories
  groups) · garage-sale utility (BST/sales groups) · zero-pitch events-only
  (groups that ban ads) · advertiser pitch (B2B groups) · spotlight (2nd group in
  a town, to avoid duplicate text).
- Sibling groups (same town): always reword fully and post on different days —
  duplicate text across groups is a spam signal.
- Every post ends with a locally-native engagement question; never "comment and
  I'll add it" (content is self-serve).
- Family-safe filter drops nightclub/adult/burlesque events from post drafts.
- To capture ALL posts: Page -> Activity log -> filter to **Posts**.
